TextUtils 

A responsive and user-friendly text utility web application built using React.js. This project helps users perform multiple text operations such as converting text to uppercase/lowercase, removing extra spaces, copying text, and counting words & characters instantly.


* Features:-

  
Convert text to Uppercase

Convert text to Lowercase

Remove extra spaces

Copy text to clipboard

Clear text instantly

Word and character counter

Reading time estimation

Dark Mode support 🌙

Responsive UI for desktop and mobile

Technologies Used

* React.js

* JavaScript

* HTML5

* CSS3

* Bootstrap


 Project Highlights

 
Developed an interactive and responsive user interface

Implemented real-time text processing functionalities

Added dark mode toggle for better user experience

Optimized performance using React state management


Purpose of the Project


This project was built to improve productivity by providing useful text formatting and analysis tools in a simple and efficient interface.
